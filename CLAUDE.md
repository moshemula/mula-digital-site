### דפוסים שכבר הוקמו — שמור עליהם
- **כל דף תוכן** = הדר עם תמונת רקע (overlay כהה + כותרת/פירורי לחם/קטגוריה/byline מעליה) + **פוטר מלא** זהה.

- אייקונים = SVG בלבד. אין אימוג'ים.
- נתיבים יחסיים ל-assets (`assets/...`).

## איך מריצים / מעלים

**הרצה מקומית** (חובה שרת — לא לפתוח index.html כ-file:// בגלל ה-router):
```bash
python3 -m http.server 8000
# או: npx serve
```
פתח `http://localhost:8000`.

**Deploy ל-Netlify:** גרור את **כל התיקייה** (לא רק index.html!) ל-app.netlify.com/drop — כולל `assets/`.
או חבר ל-Git ו-Netlify יבנה אוטומטית (אין build command, publish dir = השורש).

**מעבר לדומיין muladigital.co.il בעתיד:** החלף `mosh123.netlify.app` בכל canonical/og/sitemap/robots.
